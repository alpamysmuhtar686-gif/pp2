Quick guide for direction:
1. Archive files: need to be ignored, this is used when testing, so this is safe back versions in case of bugs or errors
2. connect.py - connection file returning a cursor
3. functions.sql - functions are used in executing
4. procedures.sql - same, but procedures
5. import_contacts.json - json files to be imported
6. database.ini - database configuration
7. schema.sql - schemas for tables
8. simple_contact.csv - simple .csv example
9. config.py - use configurations to connect with DB
10. phonebook.py - main executable file
